# Trixy

Trixy is a smart event based multi-calendar app. A very early WIP

## Installation

### Prerequesites

- Node && NPM
- Ionic cli >=5
- Firebase tools

You can follow the official installation guide for each prerequesite.

```
git clone
npm i
firebase login
ionic build --prod --release
firebase deploy
```

## Stack

- Ionic
- Firebase Authentification
- Firebase Database
- Firebase Functions
